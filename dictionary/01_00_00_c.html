clamps

(clamps &key (gui-root "/tmp/") (start-gui t))

Start clamps including the gui

gui-root ist the path where to put the files accessible by the gui. In the given path the following directories will be created:

  • <gui-path>/www/svg

    file path for svg files used in the svg-display page of the gui.

    Any files which need to be accessible by the gui have to be put into the <path>/www subdirectory with their filenames relative to this directory.
